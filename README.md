一个简洁的 ChatGPT 客户端

# 运行

https://www.555913333.xyz/

## 本地运行

`npm install`
`npm run dev`

浏览器转到`localhost:3000`

# 使用

1. 设置 OpenAI API Key
2. 在输入框中输入问题，点击发送

# 模型

对话:GPT-3.5 turbo
图片生成:DALL-E

## 说明

默认 api 地址为 cloudflare works 构建的地址

不会上传任何用户数据,apikey,api,temprature 和对话记录都存储在浏览器 localstorage 中

如果遇到 ai 胡言乱语的问题,把对话随机性适当下调

openai 生成的图片在服务器具有有效期,如果需要请保存到设备中
